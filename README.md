## React-Browserify Boilerplate

A gulp/browserify starter kit for react, with a few bells and whistles, including
* es6 transpilation
* linting
* live reload
* scss compilation
* image minification
* deployment to gh-pages


### Installation
```bash
npm install -g gulp
npm install
```

### Development

```bash
npm run dev
```

### Deployment

```bash
npm run deploy
```
